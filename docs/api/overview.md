# API Overview

The API is served under `/api/v1` and uses OpenAPI 3 schemas generated by `flask-openapi3`.

## Groups

- `GET /api/v1/health`
- `GET /api/v1/liveness`
- `GET /api/v1/readiness`
- `GET /api/v1/metrics`
- `/api/v1/auth`
- `/api/v1/projects`
- `/api/v1/forms`
- `/api/v1/sections`
- `/api/v1/questions`
- `/api/v1/choices`
- `/api/v1/actions`
- `/api/v1/conditions`
- `/api/v1/rate-limits`
- `/api/v1/ui-templates`

## Compatibility

Legacy routes under `/api/...` redirect to the `/api/v1/...` equivalents where compatibility is still maintained.
