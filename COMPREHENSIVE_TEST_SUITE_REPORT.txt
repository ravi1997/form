RUNNING THE TESTS
================================================================================

INSTALLATION:
  pip install -r requirements-test.txt

BASIC COMMANDS:
  pytest tests/ -v                  # Run all tests
  pytest tests/ --cov=app           # With coverage
  pytest tests/ -m security -v      # Security tests only
  pytest tests/ -m integration -v   # Integration tests only

SPECIFIC TEST FILE:
  pytest tests/test_models_user.py -v
  pytest tests/test_services_auth.py -v
  pytest tests/test_api_auth.py -v

SPECIFIC TEST:
  pytest tests/test_models_user.py::TestUserBasicCreation::test_create_user_with_minimal_fields -v

DEBUGGING:
  pytest tests/ -s                  # Show print output
  pytest tests/ -x                  # Stop at first failure
  pytest tests/ --pdb               # Debugger on failure
  pytest tests/ --tb=long           # Full traceback

PERFORMANCE:
  pytest tests/ --durations=10      # Show slowest tests
  pytest tests/ -n auto             # Parallel execution
  pytest tests/ --lf                # Run last failed

COVERAGE REPORT:
  pytest tests/ --cov=app --cov-report=html
  # Open htmlcov/index.html in browser
